# Security Policy

## Supported Versions

The following versions of Prescripto are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

## Reporting a Vulnerability

We take the security of Prescripto seriously. If you find a security vulnerability, please do **NOT** open a public issue. Instead, report it privately.

Please report security vulnerabilities by emailing the maintainers. We will acknowledge your report and coordinate a fix/disclosure plan.
